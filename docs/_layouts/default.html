How does a seller get put on this list?

We review feedback for eBay sellers. If there is any feedback from a buyer about fake grips, the seller gets added to this list. We save data about the seller to track any new accounts the seller might create.

What can I do if I suspect the grips I purchased are fake?

Golf Pride suggests price is the determining factor. If the grips you purchased are significantly discounted, chances are the grips are counterfeit.
